|-----------|---------|-----|

| Compliance Report Engine | Automated compliance evidence generation | `ComplianceReport` |
| Holds API | Legal/compliance hold on data deletion | `Hold` |
| Airgap Packer | Offline artifact bundling (`aikctl airgap pack`) | вҖ” |

## Key Design Decisions

1. **Cedar over OPA** вҖ” Cedar provides strongly-typed, analyzable policies with formal verification support. Deployed as a sidecar or standalone service; OPA PDP remains as fallback.

2. **Federation Mesh** вҖ” Peer-to-peer gossip for resource metadata sync. Hub-spoke mode available for strict network topologies.

3. **Marketplace OCI Registry** вҖ” Skill packs are OCI artifacts. Reuses existing container registry infrastructure and tooling.

4. **Compliance as Code** вҖ” Report definitions are CRDs. Evidence collection is automated via audit log queries and policy evaluation snapshots.
